## **Чат-бот для отслеживания выполнения условий челленджей с кружочками**

https://t.me/vchallengebot

**Особенности работы**
- возможность самостоятельно записаться или выписаться из участия в челлендже по нажатию соответствующих комманд `/start` и `/sign_out`
- видеозаметка не принимается, если пользователь не отправил команду `/start`
- выполняется проверка длительности видеозаметки (>59 секунд)
- при выполнении условий в базе создается соответствующая запись с указанием чата, пользователя и даты последней активности
- участники, отправившие команду `/start`, но не приславшие в течение дня видеозаметку, попадают в список должников
- получение списка должников по команде `/stat_of_day` (доступна только администраторам и владельцу канала). Предварительно необходимо отправить боту в ЛС команду `/start`, чтобы бот имел возможность отправлять отчеты
- отправка соответствующих комментариев на различные события, связанных с принятием участия, выходом, валидных и невалидных кружочков.
